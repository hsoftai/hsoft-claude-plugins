secrets-guard 0.8.0: env-injection model, guided onboarding, no sandbox, inert in Cowork

Architecture simplification to two commands (the hook-injected wrapper and the
model-facing `secrets-guard run`) plus guided onboarding.

Resolution / leak fixes:
- Bash command references are provisioned via the child ENVIRONMENT, not injected
  inline: the hook replaces each reference with a ${SG_REF_n} placeholder and wraps
  the command in `SG_REF_n='<ref>' secrets-guard run -- sh -c '<cmd>'`. The value
  never enters the command text, argv, the shell, the transcript, or disk, so it no
  longer reaches Claude Code's permission classifier.
- `secrets-guard run` is allowed for the model (env-only injection, output stays
  redacted); `secrets-guard read` and the raw vault CLIs remain denied.
- PostToolUse blocks a confirmed vault-value leak instead of relying on the
  unreliable updatedToolOutput.

Onboarding / install:
- `secrets-guard install` now installs the Keeper CLI if missing (winget/brew/pip),
  prompts for a one-time token, runs profile init, and validates the connection.
- require_vault (default on): a prompt is blocked with Keeper setup instructions
  when no vault is configured; set off to allow use without a vault.

Resolver robustness:
- A trailing unbalanced ] is no longer swallowed into a reference; balanced
  [index] notation is preserved.
- A single unresolvable reference no longer aborts the whole command.

Removed / scope:
- Removed the sandbox feature (subcommand, in-place rendering, journal, and the
  sandbox/sandbox_globs/kernel_dlp/dlp_install_source options).
- secrets-guard is INERT in Cowork (operates only in Claude Code local); the block
  and all enforcement never affect Cowork or claude.ai.

Validated against a real Keeper vault: ~50 adversarial checks across encodings,
Read/Write, command injection, CLI denial, prompt blocking, guard fail modes,
backslash escapes, and Cowork inertness; plus real `claude -p` sessions. Full Go
test suite, go vet, and cross-compile (windows/darwin/linux) green. Binaries built
at 0.8.0.
